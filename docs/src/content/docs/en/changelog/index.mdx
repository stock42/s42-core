## [1.0.3] - 2025-01-02
### Added
- Soporte para SQLite.
- Método `RouterController.addGlobal` agregado.

### Fixed
- Corrección de errores en el control de callbacks del servidor.

---

## [1.0.2] - 2024-12-15
### Added
- Introducción de Server-Sent Events (SSE) para soporte en tiempo real.

### Fixed
- Mejor manejo de errores en eventos de dominio.

---

## [1.0.1] - 2024-12-01
### Added
- Soporte para Event Domains con Redis.
- Clusterización para entornos escalables.

### Fixed
- Ajustes en los encabezados de respuesta predeterminados.

---

## [1.0.0] - 2024-11-01
### Added
- Lanzamiento inicial de s42-core con soporte para microservicios y arquitecturas basadas en células.
- Módulos para manejo de rutas dinámicas, controladores y eventos.
